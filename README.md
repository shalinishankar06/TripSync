# TripSync ✈️🌍
### Smart Collaborative Travel Planning Platform

TripSync is a modern full-stack travel planning and expense management application designed to simplify group trip organization. It helps users collaboratively plan itineraries, manage budgets, track expenses, assign tasks, and organize travel details — all in one place.

Built with modern web technologies including React, TypeScript, Tailwind CSS, and a scalable API architecture, TripSync delivers a smooth and responsive user experience for travelers and teams.

---

# 🚀 Features

## 🧳 Trip Management
- Create and manage multiple trips
- Add destinations, travel dates, and trip descriptions
- Upload custom cover images for trips
- View active, planning, and completed trips

---

## 📅 Itinerary Planning
- Organize activities day-by-day
- Add hotels, transport, restaurants, and activities
- Schedule timings and locations
- Keep all travel plans in one timeline

---

## 💰 Expense & Budget Tracking
- Set trip budgets
- Track expenses in real time
- Categorize spending:
  - Accommodation
  - Food
  - Transport
  - Activities
  - Shopping
  - Others
- Monitor remaining budget dynamically
- Expense analytics and summaries

---

## 👥 Team Collaboration
- Add trip members
- Assign roles and responsibilities
- Manage shared planning collaboratively

---

## ✅ Task Management
- Create trip-related tasks
- Assign priorities:
  - Low
  - Medium
  - High
- Track task completion status
- Monitor pending tasks

---

## 🌎 Currency Converter
- Built-in currency conversion utility
- Helpful for international travel budgeting

---

## 🌙 Modern UI/UX
- Fully responsive design
- Beautiful modern card-based UI
- Dark mode support
- Smooth animations and transitions
- Mobile-friendly experience

## ⚙️ Installation & Setup

# Clone Repository

- git clone https://github.com/shalinishankar06/TripSync.git

# Navigate to Project

- cd TripSync

# Install Dependencies

- pnpm install
 
# Run Development Server

- pnpm dev

# 🏗️ Build Project

- pnpm build
